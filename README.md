# ルーレット形式質問者決定アプリ (Roulette Selector)

勉強会や発表会で、参加者の中からランダムに質問者を選出するためのWebアプリケーションです。
高級感のあるモダンなUIと、スムーズなアニメーションで抽選を盛り上げます。

![Screen Shot](./candidate_list_bottom_1766661618025.png)
*(※画面は開発中のものです)*

## ✨ 主な機能

1.  **直感的なCSVインポート**
    *   参加者リストをCSVファイルでドラッグ＆ドロップするだけで準備完了。
    *   データはブラウザ内でのみ処理され、外部サーバーには送信されないためセキュアです。

2.  **登壇者ごとのフィルタリング**
    *   CSVに「宛先（登壇者名）」を含めることで、その発表者に対する質問者候補のみを自動でフィルタリングします。
    *   タブ切り替えでスムーズに対象者を変更可能です。

3.  **事前除外機能**
    *   特定の参加者をあらかじめ抽選対象から外しておくことが可能です（CSVで指定）。
    *   画面上で手動でチェックを外すこともできます。

4.  **候補者検索**
    *   多くの参加者がいても、検索バーから名前で絞り込み、特定の人をすぐに探せます。

5.  **レスポンシブデザイン**
    *   PCでは「リスト」と「ルーレット」の2カラム表示で効率的に。
    *   スマホやタブレットでは縦並びに見やすく自動調整されます。

## 📝 入力データの形式 (CSV)

以下のフォーマットのCSVファイルを用意してください。文字コードは **UTF-8** 推奨です。
ヘッダー行（1行目の見出し）はあってもなくても動作します。

| 列番号 | 項目名 | 必須 | 説明 |
| :--- | :--- | :--- | :--- |
| 1 | **名前** | 必須 | 参加者の名前 |
| 2 | **質問内容** | 必須 | 質問のテキスト（改行可） |
| 3 | **宛先** | 任意 | 質問対象の登壇者名（空欄の場合は `All` 扱い） |
| 4 | **除外コード** | 任意 | `x` または `*` を入力すると、初期状態で**除外**（チェックOFF）になります |

**CSV記述例:**
```csv
山田 太郎, 〇〇の件について質問です, 田中部長,
佐藤 花子, デザインの意図は何ですか？\nまた、配色は？, 鈴木リーダー,
除外 ユーザー, (今回は見学), , x
```

## 🚀 使い方

1.  `index.html` をブラウザで開く（またはGoogle Apps ScriptとしてデプロイしたURLにアクセス）。
2.  作成したCSVファイルをアップロードエリアにドラッグ＆ドロップ。
3.  画面右下の「**Target Presenter**」タブを選択し、抽選対象のグループを表示。
4.  「**START**」ボタンをクリックしてルーレットを回転！
5.  厳正な抽選の結果が表示されます。

## 🛠️ 開発・デプロイ

### ローカルでの実行
リポジトリをクローンし、`index.html` を直接ブラウザで開くだけで動作します。

### Google Apps Script (GAS) へのデプロイ
`index_gas.html` の内容を使用します。
Google Apps Scriptプロジェクトを作成し、HTMLファイルとしてこのコードを貼り付け、`doGet` 関数を設定してWebアプリとして公開してください。
