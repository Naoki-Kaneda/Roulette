# ルーレット形式質問者決定アプリ (Roulette Selector)

勉強会や発表会で、参加者の中からランダムに質問者を選出するためのWebアプリケーションです。
高級感のあるモダンなUIと、スムーズなアニメーションで抽選を盛り上げます。

![Screen Shot](./candidate_list_bottom_1766661618025.png)
*(※画面は開発中のものです)*

## ✨ 主な機能

1.  **柔軟な参加者登録**
    *   **CSVインポート**: リストをドラッグ＆ドロップで一括読み込み。
    *   **手動入力**: 画面上のタブから名前と質問を直接入力して追加可能。
    *   **実行中の追加**: ルーレット画面のサイドパネルから、いつでも新しい参加者を追加できます。

2.  **登壇者ごとのフィルタリング**
    *   CSVに「宛先（登壇者名）」を含めることで、その発表者に対する質問者候補のみを自動でフィルタリング。
    *   タブ切り替えでスムーズに対象者を変更可能です。

3.  **同時当選機能 (1〜3人)**
    *   一度のルーレットで最大3人まで同時に選出可能。
    *   人数に合わせてルーレットの矢印が動的に変化します。

4.  **事前除外 & 候補者検索**
    *   CSVでの事前除外フラグに対応。
    *   検索バーで特定の候補者を即座に見つけて個別に除外/復帰。

5.  **高度なテキスト表示**
    *   **改行の保持**: CSVデータや手動入力に含まれる改行を、当選画面でもそのまま表示。
    *   **長文スクロール**: 質問が非常に長い場合でも、レイアウトを崩さずスクロールで全文確認可能。

6.  **アクセシビリティ & キーボード操作**
    *   当選画面やヘルプ画面は、**Enterキー** または **Escキー** でサクサク閉じることが可能。
    *   マウスなしでもテンポよくイベントを進行できます。

7.  **プレミアム・レスポンシブデザイン**
    *   ダークモードとガラスモーフィズムを採用した高級感のあるUI。
    *   PCでは効率的な2カラム、スマホでは見やすい縦並びに自動調整。

## 📝 入力データの形式 (CSV)

以下のフォーマットのCSVファイルを用意してください。文字コードは **UTF-8** 推奨です。

| 列番号 | 項目名 | 必須 | 説明 |
| :--- | :--- | :--- | :--- |
| 1 | **名前** | 必須 | 参加者の名前 |
| 2 | **質問内容** | 必須 | 質問のテキスト（改行可） |
| 3 | **宛先** | 任意 | 質問対象の登壇者名（空欄の場合は `All` 扱い） |
| 4 | **除外コード** | 任意 | `x` または `*` を入力すると、初期状態で**除外**になります |

## 🚀 使い方

1.  **参加者を登録する**:
    *   CSVファイルを読み込むか、「手動入力」タブから名前と質問を入力します。
2.  **ルーレットを開始する**:
    *   「ルーレットを開始」ボタンを押してメイン画面へ。
3.  **設定と抽選**:
    *   「Target Presenter」タブで対象グループを選択し、必要であれば「当選人数」を設定します。
4.  **START!**:
    *   ボタンをクリックしてルーレットを回転！結果がモーダルで表示されます。
5.  **後からの追加**:
    *   急な参加者がいても、左側の「参加者の追加」パネルからいつでも追加可能です。

## 🛠️ 開発・デプロイ

### ローカルでの実行
リポジトリをクローンし、`index.html` をブラウザで開くだけで動作します。

### Firebase Hosting
本プロジェクトは Firebase Hosting に最適化されています。
```bash
firebase deploy --only hosting
```
